## cluster environment
    hadoop 2.7.1 (yarn)
    java 1.6.0_17
    CPU: Xeon (R) E5-2620
    Memory: 64GB
    Disk: 6TB
    Network: 1 Gbps Ethernet

    Lab3: Lab about HBase and Hive
